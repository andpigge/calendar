# Календарь на ванильном js. Перенесу на React
## Календарь будет служить для задачи, планирования своих задач. Также подойдет для расписания графика на работе.
## Проект в разработке

### Описание (функциональность) :
* Календарь расписан по дням. Если нажать на любой день, откроется попап где можно ввести задачу на сегодня, а также описание. В задаче можно написать имя, а в описании саму суть задачи.
* Все задачи отправляются на сервер с помощью express, и хранятся в mongoDB.

### Использовались технологии:
#### __Для верстки__:
* В разработке;
#### __Для javascript__:
* React
* mongoDB
* express
